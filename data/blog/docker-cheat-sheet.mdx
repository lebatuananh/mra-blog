## Cài đặt

### Linux

Xem chi tiết cách cài đặt tại [đây !](https://docs.docker.com/get-docker/)

```
curl -sSL https://get.docker.com/ | sh
```

### Mac

Xem chi tiết cách cài đặt tại [đây !](https://docs.docker.com/docker-for-mac/install/)

Sử dụng đường dẫn này để tải file dmg, sau đó mở file đã tải và cài đặt như hướng dẫn

```
https://download.docker.com/mac/stable/Docker.dmg
```

### Windows

Xem chi tiết cách cài đặt tại [đây !](https://docs.docker.com/docker-for-windows/install/)

Sử dụng đường dẫn này để tải file msi, sau đó mở file đã tải và cài đặt như hướng dẫn

```
https://download.docker.com/win/stable/InstallDocker.msi
```

## Docker Registries & Repositories

### Login Registry

```
docker login
docker login localhost:8080
```

### LogOut Registry

```
docker logout
docker logout localhost:8080
```

### Searching image

```
docker search nginx
docker search --filter stars=3 --no-trunc nginx
```

### Pull Image

```
docker image pull nginx
docker image pull eon01/nginx localhost:5000/myadmin/nginx
```

### Pushing Registry

```
docker image push eon01/nginx
docker image push eon01/nginx localhost:5000/myadmin/nginx
```
